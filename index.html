How to use html and Css on Vs Code

Steps

  1. Create a file on your desktop or any other directory
  2. Open your editor then click open folder( select the folder you created in your desktop)
  3. It will display on your editor, the create a file on it and name it index.html
  4. Create a folder on your parent file and name it Resources
  5. Create a file on resources and name it index.css
  6. In the editor on index.html start by typing a document declaration
    (!DOCTYPE html)
  7. Then type the document type
    <html></html>
    which will enclose the entire document
  8. Then within <html></html>thats where the < title></title> and the body element tag goes to
    what is in the title tag appears in the browser while what is in the body appears in the page(window)
  9. Then link the index.css file you created to your code index.html by inserting a link element tag on the
    head element of the document as follows
    <link rel="stylesheet" href="resources/index.css">
  10. Write your code inside the body element tag
  11. For you to create code in the index.css sheet, create selectors from the index.html codes i.e identifyers
    like classes and id to work with in the index.css file. you can also use selector of elements in the index.html
    file e.g <h1> and write it as h {} in css,
    if you give an element a class eg <h1 class="header">,in the css file the selector will be
    .header {} for styling of all the contents in the <h1> element. You can also make ids which which are used
    when describing very specific instruction to to elements that you want to be unique in the documents
    despite being within common element tags
